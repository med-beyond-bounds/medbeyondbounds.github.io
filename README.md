# Med Beyond Bounds Website

Static 4-page website for Med Beyond Bounds, designed for free hosting on GitHub Pages.

## Pages

- `index.html` (Home)
- `mission.html` (Our Mission)
- `get-involved.html` (Get Involved)
- `committees.html` (Our Committees)

## Free Hosting Setup (GitHub Pages)

1. Push these files to your GitHub repository.
2. In GitHub: `Settings` -> `Pages`.
3. Under `Build and deployment`, set:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` (or your default branch), folder `/ (root)`
4. Save. GitHub will publish your site at:
   - `https://<username-or-org>.github.io/<repo-name>/` for project pages, or
   - `https://<username-or-org>.github.io/` for user/org pages (repo name must exactly match).

## Make It Google Searchable

1. Keep `robots.txt` and `sitemap.xml` in the root.
2. Create a Google Search Console account.
3. Add your website property URL exactly as published.
4. Submit `https://your-site-url/sitemap.xml` in Search Console.
5. Optionally request indexing for your main pages.
